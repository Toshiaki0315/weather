# 気象庁API 天気予報取得スクリプト (JMA Weather CLI)

気象庁が提供している公式のJSONデータから、指定した市区町村の天気予報を取得してターミナルにわかりやすく表示するPythonスクリプトです。
将来的なRaspberry Piでの常駐稼働（スマートディスプレイ化）を見据え、堅牢なデータ取得ロジックとロギング機能を備えています。

## ✨ 主な機能

* **市区町村単位でのスマートフィルター**
  気象庁のデータは「地方（東部・西部など）」単位ですが、本スクリプトでは入力された「市区町村（例：横浜市、小田原市）」が属する予報区を自動で割り出し、必要なエリアの情報だけをピンポイントで表示します。
* **リッチな予報データの取得**
  今日・明日の天気、6時間ごとの降水確率、予想気温（最低・最高）、および向こう3日間の週間天気を一覧表示します。
* **堅牢なデータパース（フォールバック機能）**
  早朝など、気象庁の短期予報データから気温が欠落する時間帯でも、週間予報データから自動的に気温を補完して表示します。
* **ロギング機能（常駐化対応）**
  実行履歴や通信エラーの詳細を `weather.log` に自動記録します。Cron等を用いたバックグラウンドでの定期実行時に役立ちます。

## 🛠 動作環境

* Python 3.6 以上
* 標準ライブラリのみを使用しているため、`pip install` などの追加パッケージのインストールは不要です。

## 🚀 使い方

ターミナルでスクリプトを実行し、引数に天気を知りたい「都道府県」または「市区町村名」を指定します。

```bash
# 基本的な使い方（引数なしの場合はデフォルトで「横浜」が検索されます）
python weather.py

# 市区町村を指定して実行（該当する予報区のみ表示）
python weather.py 横浜市
python weather.py 小田原市

# 都道府県を指定して実行（県内すべての予報区を表示）
python weather.py 神奈川県
```

### 出力サンプル

```text
$ python weather.py 横浜市
「横浜市」の天気予報取得を開始します。

========== 【横浜市の天気予報】 ==========
■ 東部
  [今日] 晴れ 時々 くもり
  [明日] くもり 昼過ぎ から 晴れ 所により 昼前 まで 雨
  [降水] 10% / 20% / 40% / 40% / 0% / 0%
  [気温] 24℃ / 24℃ / 17℃ / 25℃
  [週間] 向こう3日間: 晴れ ➔ くもり ➔ 雨
----------------------------------------
```

## 🧪 テストの実行

外部API（気象庁サーバー）に負荷をかけずにロジックを検証するため、`unittest.mock` を使用した単体テストを用意しています。

```bash
python -m unittest discover tests
```
